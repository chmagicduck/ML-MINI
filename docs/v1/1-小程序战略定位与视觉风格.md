# 吗喽薪事 (Malou Slacker) - 产品战略规格书

## 1. 产品定义
- **核心定位**：一款专为打工人设计的“反卷”可视化工具，将工作时间转化为金钱收益。
- **核心价值**：通过实时计算“带薪摸鱼”收益，缓解职场焦虑，提供情绪价值。
- **目标用户**：追求职场 WLB（工作生活平衡）、喜欢玩梗、对薪资结构敏感的 90/00 后职场人。

## 2. 品牌基因
- **产品名称建议**：吗喽薪事
- **Slogan**：只要我不努力，老板就永远过不上想要的生活。
- **视觉风格 (Design Language)**：
    - **色调**：采用“护眼绿”为主色（#E8F5E9），配合温和的卡片投影（Soft UI）。
    - **调性**：反差萌。用最严谨的金融图表风格，展示最荒诞的“摸鱼数据”。
    - **核心元素**：吗喽（猴子）Emoji/插画、实时跳动的数字、极简主义卡片。

## 3. 商业闭环建议 (后期)
- **社交激励**：生成“摸鱼战报”分享朋友圈（裂变）。
- **成就系统**：解锁“摸鱼王者”、“拉屎达人”等勋章。
- **功能扩展**：接入“老板来了”紧急避险模式（一键切换到 PPT/Excel 伪装页）。
