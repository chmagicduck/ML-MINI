# 页面规格说明书 (适用于 Claude Code 生成)

## 页面 A：首页 (Dashboard) - `pages/index/index`

### 1. 核心逻辑 (Business Logic)
- **收益计算公式**：
    - 日薪 $D = \frac{月薪}{当月工作日}$
    - 秒薪 $S = \frac{D}{(下班时间 - 上班时间 - 午休时长) \times 3600}$
    - **今日摸鱼收益** = $秒薪 \times 累计摸鱼秒数$
- **计时器逻辑**：点击中央圆圈开启/关闭“摸鱼模式”。开启后，收益数字实时跳动（$100ms$ 刷新率）。

### 2. 视觉组件规格
- **顶部状态栏**：设置图标 (左)、小程序胶囊 (右)。
- **中央仪表盘**：
    - 圆形进度条：展示今日已流逝的工时比例。
    - 状态标签：红色 Pill 标签显示“摸鱼模式”。
    - 计时器：`HH:mm:ss` 格式，大号数字字体。
- **功能卡片阵列 (2xN Grid)**：
    - **今日摸鱼收入**：实时更新的金额。
    - **本月收入**：实时更新的本月已赚金额。
    - **距离发薪日**：基于 `Settings` 中的发薪日计算倒计时。
    - **倒计时卡片**：距离周日、退休日的动态计算。
    - **特色入口**：
        - `带薪拉粑粑`：点击进入独立计时器，记录年累计“拉屎收益”。
        - `今天吃什么`：随机抽奖组件（解决职场世纪难题）。

## 页面 B：设置主页 (Settings) - `pages/settings/index`

### 1. 交互列表
- **列表项 (Cell)**：左侧图标 + 标题 + 副标题 + 右侧箭头。
- **主要配置项**：
    - `基本设置`：薪资、工时配置入口。
    - `重要日期`：入职、退休日配置。
    - `特效设置`：摸鱼时是否播放“金币掉落”音效/动画。
    - `主题设置`：切换配色方案。

---

## 页面 C：基本设置 (Basic Config) - `pages/settings/basic`

### 1. 数据采集单 (Form)
- **薪资配置**：
    - 月薪输入框（Type: Digit）
    - 发薪日期选择器（1-31号）
- **时间配置**：
    - 上班/下班时间（Time Picker）
    - 午休/晚休开关 + 时间段选择。
- **工作日设置 (Workday Picker)**：
    - 快捷选项：双休、周六休、周日休、大小周。
    - **自定义日历**：允许用户点击日历勾选本月加班日或调休日。

### 2. 数据持久化
- 所有配置项需存储于 `wx.setStorageSync`，并在首页 `onShow` 时重新初始化计算引擎。
