# 吗喽摸鱼助手 (Malou Slacker) V2.0 升级规格说明书

## 一、 产品战略与架构 (Strategic Mapping)

### 1.1 核心理念升级
从单纯的“工时计算器”进化为“打工人情绪价值中心”。通过增加“等级系统”和“会议烧钱逻辑”，提升产品的传播属性。

### 1.2 导航架构 (Tabbar)
- **首页 (pages/index/index)**：核心计时器、实时收益、功能卡片入口。
- **战报 (pages/stats/index)**：累计数据、摸鱼热力图、等级成就。
- **我的 (pages/profile/index)**：薪资设置、节假日日历、关于我们。

## 二、 核心逻辑优化 (Technical Logic)

### 2.1 解决后台计时/杀进程问题 (Timestamp Strategy)
- **逻辑**：不使用 `setInterval` 作为唯一计数源，改用**时间戳差值法**。
- **流程**：
  1. 用户点击“开始摸鱼”，`wx.setStorageSync('moyu_start_time', Date.now())`。
  2. 页面 `onShow` 时，计算 `diff = Date.now() - moyu_start_time`。
  3. 即使小程序被杀掉，只要缓存还在，重新打开时收益会自动同步最新差值。
  4. 收益公式：`今日收益 = (diff / 1000) * 秒薪`。

### 2.2 本地节假日引擎
- **实现**：在 `utils/holiday.js` 中内置 2026 年法定节假日和调休数据。
- **功能**：
  - 自动判断今日是否为“法定加班日”或“带薪节假日”。
  - 若为周末或节假日，首页显示“休息中，目前是纯利润时间”。

## 三、 页面详细功能规格

### 3.1 首页 (Home) - 极致爽感增强
- **数字同步**：`今日收益` 与 `本月收益` 必须保持相同的 `100ms` 刷新频率，让用户感到钱在源源不断流入口袋。
- **新增卡片：会议烧钱机**
  - 入口：首页显著卡片。
  - 功能：用户选择“参会人数”，点击开始。屏幕实时跳动“本场会议已消耗公司：¥XX.XX”。
- **新增卡片：吗喽等级**
  - 根据累计收益自动升级。
  - 级别定义：
    - < 500: 职场牛马
    - 500 - 2000: 摸鱼学徒
    - 2000 - 10000: 带薪锦鲤
    - 10000+: 划水宗师
    - 50000+: 摸鱼大圣 (获得专属金色边框)

### 3.2 统计页 (Stats) - 数据可视化
- **累计财富**：展示自安装以来累计赚取的总额。
- **摸鱼热力图**：使用 Canvas 绘制类似 GitHub 的绿色方块矩阵，展示每日摸鱼活跃度。
- **分享功能**：点击“生成战报”，合成一张包含“吗喽文学语录”和“摸鱼等级”的海报。

### 3.3 设置与日历 (Settings & Calendar)
- **日历视图**：点击“距离发薪/周末”卡片跳转。
  - 标注出发薪日、周末、法定节假日。
  - 底部显示文案：“坚持住，再过 X 个工作日就能发薪了！”

## 四、 视觉与交互规范
- **色彩**：延续护眼绿 (#E8F5E9)，增加金色 (#FFD700) 作为高级等级的辅助色。
- **动效**：
  - 数字跳动伴随轻微的 `wx.vibrateShort` (用户可配置开关)。
  - 切换摸鱼模式时，吗喽头像从“打工苦瓜脸”变为“闭眼享受脸”。
