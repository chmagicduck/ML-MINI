# 🌊 品牌名称：薪潮涌动 (Salary Tide)

「薪潮涌动」是一款专为职场“潜水员”打造的带薪摸鱼小程序，将枯燥工时化作实时跳动的深海资产，让你像看理财产品一样围观薪水涨潮，精准捕捉每一秒划水带来的“鱼粮”收益，在蓝海洋流中优雅计薪，感受带薪入账的快感。

## 1. 核心品牌语 (Slogan)
“感受每一秒薪水的涨潮。”
(副标题：让打工的时间，变成翻涌的资产。)

## 2. 品牌定位 (Positioning)
职场潮汐能守望者。
它不只是一个计薪工具，更是一个 **“情绪转换器”**。它把原本枯燥、被动、被“买断”的时间，转化为一种像潮汐一样有规律、有力量、不断叠加的个人资产，让用户在摸鱼时感受到一种“丰收”的掌控感。

## 3. 视觉与语境调性 (Brand Tone)

- 色调建议： 延续你现有的 深海蓝 搭配 荧光青/金。蓝色代表职业与深邃，金色代表每一秒都在入账的“鱼粮”。
- 关键词： 动态、精准、暗爽、优雅。
- 语境设定： 将职场比作一片巨大的海洋，老板是季风，任务是浪头，而你只需要稳稳地坐在你的**「薪潮」**上，看水位一点点升高。

## 4. 模块化功能重命名（对齐品牌感）

为了配合“薪潮涌动”这个大名字，建议将你 UI 里的部分词汇进行“海洋化”升级，增加沉浸感：

| 原模块名称 | 建议新名称 | 品牌含义|
|---|---|---|
| 摸鱼中心 | 领潮中心   | 站在薪水涨潮的最前线 |
| 情报中心 | 洋流情报   | 观测职场动向，把握划水时机 |
| 实验室   | 深海动力室 | 研发更先进的“捕鱼”装置 | 
| 下班进度 | 退潮倒计时 | 离重返陆地（下班）还有多久 | 
| 实时流速 | 潮汐压强   | 每一秒薪水拍打口袋的力量感 |

## 5. 用户心理暗示

- 当用户打开小程序： 他不是在偷懒，他是在**“观测潮汐”**。
- 当用户盯着 ¥0.0151/秒： 他不是在发呆，他是在**“采集动能”**。
- 品牌性格： 拒绝苦大仇深，用一种**“优雅的贪婪”**去解构打工。
